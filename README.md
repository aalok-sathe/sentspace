## About The Project

The goal is for a user to feed the toolbox a csv/txt of sentences, and for the toolbox to return a csv with sentence feature values for each sentence.

### Built With

* [python3](https://www.python.org/downloads/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/gretatuckute/sent-spaces-comp.git
   ```
2. Create virtual environment and install requirements
   ```sh
   conda create -n <your_env_name> --file requirements.txt
   ```
3. Start Jupyter notebeook
   ```JS
   jupyter notebook
    ```
  Now open sent-spaces-comp/feature_emb_Sep2020.ipynb for the demo.
